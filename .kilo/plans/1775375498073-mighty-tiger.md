# Plan: Reorderable Legal Article Menu via CMS

## Current State

- Legal articles are Markdown files in `content/legal/` with YAML frontmatter containing `menuOrder` (integer) and `menuSection` (legal/payments/company).
- `content-loader.ts:77` sorts all pages by `menuOrder` ascending.
- Navigation (`legal-navigation-sheet.tsx`) renders pages per section in sorted order.
- Decap CMS exposes `menuOrder` as a plain number widget — admins must manually enter numbers, which is unintuitive.
- Decap CMS is loaded from CDN (`decap-cms@3.1.11` in `public/admin/index.html`). Custom widgets can be registered inline using `createClass` and `h` (React.createElement).

## Approach: Both Improved Hints + Custom Widget

Add descriptive `hint` to both `menuOrder` and `menuSection` fields:

Changes: increase `max` from 20 to 50, add `hint` to both fields.

## Step 2: Create custom "menuOrder" widget with visual reordering

**File:** `public/admin/index.html` (add inline script after Decap CMS load)

The custom widget will replace the plain number input with an enhanced UI showing:
- Current position indicator (e.g., "Pozycja 3 z 7 w sekcji legal")
- Up/Down arrow buttons to move the article higher/lower in its section
- The underlying `menuOrder` number is still stored in frontmatter

**Widget implementation sketch** (inline in `index.html` using `createClass` + `h`):

```js
// Register after CMS loads
CMS.registerWidget('menuOrder', createClass({
  handleChange: function(e) {
    this.props.onChange(parseInt(e.target.value, 10) || 1);
  },

  render: function() {
    var value = this.props.value || 1;
    var field = this.props.field;
    var hint = field.get('hint') || '';

    return h('div', { style: { display: 'flex', flexDirection: 'column', gap: '8px' } },
      h('label', {
        style: { fontWeight: 'bold', fontSize: '14px' },
      }, 'Kolejność w menu'),
      h('input', {
        id: this.props.forID,
        className: this.props.classNameWrapper,
        type: 'number',
        min: 1,
        max: 50,
        value: value,
        onChange: this.handleChange,
        style: { width: '100px' },
      }),
      hint ? h('span', { style: { color: '#666', fontSize: '12px' } }, hint) : null
    );
  }
}));
```

**Important limitation:** Decap CMS widgets operate on a single entry at a time. A widget cannot read other entries in the collection to show "3 of 7" context. The up/down buttons would only increment/decrement the `menuOrder` number. This is still better UX than a bare number field.

## Step 3: Update README documentation

**File:** `README.md`

Add a section to the CMS/legal pages area:

```markdown
### Reordering Legal Pages

Pages are ordered within each menu section (`legal`, `payments`, `company`) by the `menuOrder` number in frontmatter.
Lower numbers appear higher in the menu. To reorder:

1. Open the CMS at `/admin/`
2. Edit the desired page
3. Change the "Kolejność w menu" field — lower number = higher position
4. Save and publish

Ensure each page in the same section has a unique `menuOrder` value.
```
